# SimPhis_DigitalTwin
# Физическая симуляция в Unity: Лабиринт

Данный проект представляет собой интерактивную физическую симуляцию, реализованную на движке Unity 6. Основная цель — демонстрация работы физического движка, систем ввода и взаимодействия объектов через коллайдеры и триггеры.

## Основные возможности:
* **Физика твердых тел:** Использование компонентов `Rigidbody 2D` для имитации гравитации и импульсов.
* **Система столкновений:** Настройка `Box Collider 2D` и `Circle Collider 2D` для предотвращения прохождения сквозь стены.
* **Зона финиша:** Реализация триггерной зоны (`Is Trigger`), которая фиксирует достижение цели и выводит текст «ПОБЕДА!».
* **Управление:** Реализовано через классическую систему ввода Unity (Input Manager).

## Управление:
* **Стрелки (Влево/Вправо) или A/D:** Перемещение шара по горизонтали.
* **Пробел (Space):** Прыжок.
